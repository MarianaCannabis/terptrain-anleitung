# Terptrain Mitglieder-Anleitung

Interaktive Mitglieder-Anleitung für die Terptrain Community-Plattform der Mariana Cannabis Social Clubs Deutschland e.V.

## 🚀 Live ansehen

**https://MarianaCannabis.github.io/terptrain-anleitung/**

## 📋 Inhalt

- 35 Kapitel mit über 50 Screenshots
- Interaktive Tabs, Akkordeons, FAQ
- Quiz, Glossar, Changelog
- Dark/Light Mode, Mehrsprachigkeit (DE/EN/TR)
- Offline-fähig (PWA)
- Lesezeichen, Notizen, Fortschrittsanzeige

## 🛠️ Deployment

Diese Anleitung läuft als statische HTML-Seite über GitHub Pages.

**Einmalige Einrichtung:**
1. Repository `terptrain-anleitung` unter MarianaCannabis erstellen
2. Diese Dateien hochladen
3. Settings → Pages → Branch: `main`, Folder: `/ (root)` → Save
4. Unter `https://MarianaCannabis.github.io/terptrain-anleitung/` erreichbar
